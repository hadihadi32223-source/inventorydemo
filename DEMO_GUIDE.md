# دليل عرض النسخة التجريبية للعميل

هذه النسخة مخصصة للعرض على GitHub Pages أو التشغيل المحلي. تعمل ببيانات تجريبية داخل المتصفح ولا تحتاج Backend.

## طريقة الدخول

في شاشة تسجيل الدخول اضغط:

**الدخول إلى النسخة التجريبية**

يدخل النظام مباشرة بصلاحية مدير النظام لمعاينة كل الصفحات.

## ما الذي يستطيع العميل معاينته؟

- لوحة التحكم والتنبيهات.
- الأصناف والبيانات الأساسية.
- الموردون مع ملف مستقل لكل مورد.
- العملاء مع ملف مستقل لكل عميل.
- مستند وارد وربطه بالمورد.
- مستند صادر وربطه بالعميل.
- التحويل بين المستودعات.
- الجرد.
- المخزون الحالي.
- حركات المخزون.
- تنبيهات الحد الأدنى Low Stock.
- التقارير مع معاينة قبل الطباعة.
- الباركود و QR ومعاينة الملصق.
- المستخدمون والصلاحيات.
- سجل النشاط.
- النسخ الاحتياطي التجريبي.

## قواعد النسخة

- لا توجد أسعار.
- لا توجد ضرائب.
- لا توجد أرباح.
- لا توجد محاسبة.
- التركيز فقط على المخزون والمستودعات.

## ملاحظات للعميل

هذه نسخة Frontend Demo توضح شكل النظام وسير العمل. الربط مع قاعدة بيانات حقيقية وBackend آمن يتم في مرحلة الإنتاج.
