# ArabicWms Demo — نظام إدارة المخزون والمستودعات

نسخة Demo احترافية لعرض نظام إدارة المخزون والمستودعات على العميل عبر GitHub Pages.

النظام عربي RTL ويعرض فكرة برنامج مخازن كامل بدون أسعار وبدون محاسبة.

## رابط الدخول التجريبي

بعد رفع المشروع على GitHub Pages، افتح رابط Pages ثم اضغط:

**الدخول إلى النسخة التجريبية**

لا يحتاج العميل إلى معرفة حسابات أو كلمات مرور.

## أهم المميزات المعروضة

- Dashboard مع إحصائيات وتنبيهات واقعية.
- إدارة الأصناف والتصنيفات والوحدات.
- الموردون مع ملف مستقل لكل مورد وكل مستند وارد باسمه.
- العملاء مع ملف مستقل لكل عميل وكل مستند صادر باسمه.
- مستند وارد ومراجعة/طباعة.
- مستند صادر ومراجعة/طباعة.
- تحويل بين المستودعات.
- الجرد.
- المخزون الحالي.
- حركات المخزون.
- Low Stock Alerts.
- تقارير مع Preview قبل الطباعة.
- Barcode / QR Labels مع معاينة وطباعة.
- مستخدمون وصلاحيات.
- سجل النشاط.

## قواعد النظام

- لا أسعار.
- لا تكلفة.
- لا ضرائب.
- لا أرباح.
- لا قيمة مخزون.
- لا محاسبة.
- يستخدم النظام مصطلحات: مستند وارد / مستند صادر.

## التشغيل المحلي

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
npm run preview
```

## رفعه على GitHub Pages

1. أنشئ Repository جديد على GitHub.
2. ارفع ملفات المشروع.
3. ادخل إلى Settings > Pages.
4. اختر GitHub Actions كمصدر النشر.
5. ادفع المشروع إلى `main`.
6. سيعمل workflow الموجود في `.github/workflows/deploy.yml` ويبني المشروع تلقائيًا.

## ملاحظات تقنية

- المشروع يعمل كـ Static Frontend Demo.
- البيانات التجريبية موجودة داخل `src/lib/mockData.ts`.
- لا يحتاج Supabase أو Backend في وضع العرض.
- تم استخدام HashRouter ليعمل بشكل صحيح على GitHub Pages.
- عند الإنتاج الحقيقي يتم ربطه بقاعدة بيانات وBackend آمن.
